# متجر باريس للهواتف والابتوبات 📱💻

متجر إلكتروني احترافي وفاخر متخصص في بيع الهواتف الذكية والابتوبات بأفضل الأسعار والجودة العالية.

## 🎨 فلسفة التصميم

**Modern Luxury Tech Minimalism** - تصميم فاخر حديث يجمع بين البساطة الأنيقة والتقنية المتقدمة

### الألوان الأساسية
- **الأساسي**: أزرق عميق (#1E3A5F) - للثقة والاحترافية
- **الثانوي**: ذهبي دافئ (#D4AF37) - للفخامة والتركيز
- **الخلفية**: أبيض نقي (#FFFFFF) - للنظافة والوضوح
- **النصوص**: أسود عميق (#0F0F0F) - للقراءة الواضحة

### الطباعة
- **العناوين**: Playfair Display - أنيقة وفاخرة
- **النصوص**: Cairo/Inter - قابلة للقراءة وحديثة

## 🚀 كيفية التشغيل

### المتطلبات
- Node.js 16+ 
- pnpm (أو npm)

### خطوات التثبيت والتشغيل

```bash
# 1. استخراج الملف
unzip paris_phone_store.zip
cd paris_phone_store

# 2. تثبيت المكتبات
pnpm install

# 3. تشغيل خادم التطوير
pnpm dev

# 4. فتح المتصفح
# الرابط: http://localhost:3000
```

### الأوامر المتاحة

```bash
# تشغيل خادم التطوير
pnpm dev

# بناء المشروع للإنتاج
pnpm build

# معاينة البناء
pnpm preview

# التحقق من الأخطاء
pnpm check

# تنسيق الأكواد
pnpm format
```

## 📝 ملاحظات تطويرية

### إضافة منتجات جديدة
```typescript
// في client/src/data/products.ts
{
  id: 'product-id',
  name: 'اسم المنتج',
  category: 'phone' | 'laptop',
  price: 1999,
  originalPrice: 2299,
  description: 'الوصف',
  specs: ['المواصفات'],
  image: 'رابط الصورة',
  rating: 4.5,
  reviews: 100,
  inStock: true,
}
```

### تخصيص الألوان
عدّل الألوان في `client/src/index.css` في قسم `:root`

### إضافة صفحات جديدة
```typescript
// 1. أنشئ الصفحة في client/src/pages/
// 2. أضفها إلى Router في App.tsx
<Route path="/new-page" component={NewPage} />
```
